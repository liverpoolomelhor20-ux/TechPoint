# TechPoint

## Descrição do Produto/Serviço
TechPoint é uma loja online de periféricos voltada para gamers, oferecendo produtos como mouse, teclado e fones de alta qualidade. O projeto simula o fluxo de compra completo, do catálogo até o checkout.

## Tecnologias Utilizadas
- HTML
- CSS
- JavaScript

## Estrutura do Projeto
- **HTML**: páginas principais do site (`home.html`, `produto.html`, `sobre.html`, `contato.html`, `carrinho.html`, `checkout.html`, `detalhes.html`)  
- **CSS**: estilos do site (`css/estilos.css`)  
- **JS**: scripts do site (`js/carrinho.js`, `js/checkout.js`, `js/script-produtos.js`, `js/produtos.js`)  
- **Imagens**: arquivos de mídia utilizados (`imagens/`)  

## Como Executar Localmente
1. Baixe ou clone o repositório:
2. Abra o arquivo `home.html` no navegador para acessar o site.
